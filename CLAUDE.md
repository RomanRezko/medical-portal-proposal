# Коммерческое предложение — Моя Наука

## О проекте

Интерактивная презентация коммерческого предложения для медицинской сети "Моя Наука". Одностраничный HTML-сайт с навигацией между секциями.

## Технологии

- **HTML5** — single-file presentation
- **Tailwind CSS** (CDN) — стилизация с кастомной палитрой `medical-*` (emerald/green)
- **Lucide Icons** — иконки
- **Inter** (Google Fonts) — шрифт
- **Vanilla JS** — scroll navigation, Intersection Observer, keyboard nav

## Структура презентации

1. **Hero** — заголовок, теги (WP/Laravel, 20+ городов, API, ФЗ-152)
2. **Why Us** — экспертиза в СНГ + ЕС, портфолио (SEO/Dev кейсы)
3. **Roadmap** — 8 фаз, 18 недель, Gantt-chart
4. **Details** — детализация 8 фаз с описаниями
5. **Pricing** — смета 3,800,000 ₽ (10 позиций)
6. **CTA** — контакты (romanrezko@gmail.com, t.me/romanrezko)
