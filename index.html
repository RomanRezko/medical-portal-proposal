Детализация работ

Детализация работ

8 фаз проекта с подробным описанием

Фаза 01

Проектирование

Нед 1-2

Закладываем фундамент проекта: анализируем бизнес-процессы, проектируем архитектуру и создаём прототипы.

  • UX-прототипы в Figma с user flow
  • Карта сайта и техническое задание
  • Архитектура базы данных
Фаза 02

Дизайн

Нед 3-4

Создаём уникальный визуальный стиль, соответствующий медицинской тематике и современным трендам.

  • Дизайн 6 типов страниц
  • UI-kit с переиспользуемыми компонентами
  • Адаптивные макеты для всех устройств
Фаза 03

Разработка

Нед 5-8

Превращаем дизайн в работающий продукт: backend, frontend и настройка CMS для удобного управления.

  • WP/Laravel backend разработка
  • Адаптивная вёрстка без конструкторов
  • Настройка CMS для контент-менеджеров
Фаза 04

Личный кабинет

Нед 9-10

Разрабатываем функционал для пациентов: от регистрации до просмотра результатов исследований.

  • Регистрация и авторизация
  • Профиль пациента и избранные врачи
  • История заказов и результатов
Фаза 05

Интеграции

Нед 11-12

Связываем сайт с внешними системами: лаборатория, геолокация, генерация документов.

  • API/SQL интеграция с БД лаборатории
  • Корзина + генерация PDF-бланков
  • Геопозиция: 20+ городов по IP
Фаза 06

Мед. специфика

Нед 13-14

Обеспечиваем соответствие всем требованиям законодательства и доступность для всех пользователей.

  • ФЗ-152 и требования Минздрава
  • Версия для слабовидящих (WCAG 2.1)
  • Политика обработки перс. данных
Фаза 07

Контент + SEO

Нед 15-16

Переносим контент и сохраняем SEO-позиции: парсинг, импорт и настройка редиректов.

  • Парсинг и импорт контента
  • SEO-переезд с 301-редиректами
  • Сохранение поисковых позиций
Фаза 08

QA + Запуск

Нед 17-18

Финальная проверка качества, устранение багов, аудит безопасности и запуск в продакшн.

  • Функциональное тестирование
  • Аудит безопасности
  • Деплой и запуск в продакшн
